Layer 0 — это шаблон проектов Интуиции. В основе шаблонизатор Pug для html-разметки, SCSS для css-стилей, jQuery для яваскрипта. Есть набор готовых компонентов сайта в виде css-фреймворка — дизайн-системы.

Куда класть картинки

Картинки кладём в папку /static/images/. Адреса картинок будут начинаться с /images/, например:

img(src="/images/poster.jpg")

Куда класть шрифты

Файлы шрифтов кладём в папку /static/fonts/, а потом подставляем название шрифта и имя файла из папки внутри файла /styles/base/_fonts.scss в миксин:

@include font-face('Open Sans', 'open-sans-regular');

Миксин выглядит так:

font-face($family, $path, $weight: normal, $style: normal)

$family — имя шрифтовой семьи, будет использоваться внутри css-стилей, чтобы включить нужный шрифт. font-family: 'Open Sans';

$path — имя файла из папки /static/fonts/ без расширения. Одновременно подключаться все расширения. хз как написать

$weight — жирность шрифта. По умолчанию normal. Для двух начертаний можно указать normal и bold. font-weight: bold;

Если начертаний больше двух, использовать цифры, например: 100, 200, 400. font-weight: 400;

$style — наклон шрифта, normal и italic. По умолчанию normal. font-style: italic;
